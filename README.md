# Anarres-DB
Proyecto para describir en una base de datos el libro y el planeta del libro de los desposeidos de Ursula K. Le Guin.

En la carpeta documentos existe un documento de hoja de calculo con dos hojas donde se va guardando información sobre el proyecto.

En la carpeta base existe los archivos de la base de datos postgres.

También existe una imagen vistadelaescuela.png donde se ve la base de datos con los archivos php que toman los datos y los muestran (index.php).

## Posdata
Existe bastante información sobre educación y anarquismo y en el libro en uno de los primeros capitulos se cuenta un suceso del presonaje principal durante sus estudios, así que podria hacerse una base de datos de una escuela en el planeta Anarres, basado en una escuela anarquista por ejemplo la escuela moderna de Barcelona (propuesta).

Se puede ver el diagrama en [dbdiagram.io](https://dbdiagram.io/d/Escuela-Odonian-68efd4ee2e68d21b41a45975).
